================================================================================
SCORING SYSTEM IMPLEMENTED
================================================================================

Points Distribution:
  10 pts: Exact score match
   7 pts: Correct winner + goal difference
   5 pts: Correct winner + winner goals
   3 pts: Correct winner only
   2 pts: Draw (not exact match)
  +5 pts: Knockout bonus

Tiebreaker Rules (in order):
  1. Total score (descending)
  2. Correct exact results
  3. Correct winners
  4. Total predictions made

QUICK START
================================================================================

1. Install dependencies:
   npm install

2. Create .env file:
   cp .env.example .env
   (Update with PostgreSQL connection string)

3. Setup database:
   npx prisma migrate dev

4. Seed test data (optional):
   npx prisma db seed

5. Start development server:
   npm run start:dev

6. Access API docs:
   http://localhost:3001/api/docs
